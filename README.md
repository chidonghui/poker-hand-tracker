# Poker Hand Tracker

德州扑克手牌记录工具 - GitHub Pages 版本

🔗 **在线访问**: https://你的用户名.github.io/poker-hand-tracker

## 快速开始

1. 点击顶部快捷按钮（AA、KK、AKs等）快速选择手牌
2. 输入盈亏金额（如 +1500 或 -500）
3. 点击"记录手牌"保存
4. 在"复盘"页面查看历史记录和统计

## 功能特点

- 📝 快速记录：支持快捷按钮和自定义输入
- 📊 数据复盘：查看历史手牌、填写EV分析
- 📈 统计图表：盈亏趋势、位置分析
- 💾 本地存储：数据保存在浏览器本地

## 技术栈

- HTML5 + CSS3 + Vanilla JavaScript
- LocalStorage 数据持久化
- 响应式设计，支持移动端
