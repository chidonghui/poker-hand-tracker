# Poker Hand Tracker - 德州扑克手牌记录工具

专为德州扑克职业牌手设计的手牌记录与复盘工具。

## 核心功能

### 1. 牌桌快速记录
简化语法，快速输入：
- `AKs+1500` = AK同花赢1500
- `AKo-500` = AK不同花输500
- `99` = 口袋对9
- `AsKc+800` = 黑桃A梅花K赢800

### 2. 复盘界面
- 图形化扑克牌展示
- 按实际盈亏/EV排序
- 填写位置(UTG/MP/CO/BTN/SB/BB)
- 填写行动(Open/3B/4B/Call/All-in)
- 填写EV(期望值)
- 添加备注

### 3. 数据统计
- 日/周/月统计
- 实际盈亏 vs EV对比
- 盈利/亏损手牌排行
- 位置统计(哪些位置盈利最多)

## EV (期望值) 说明

**实际输赢** ≠ **EV**

- 实际输赢：你实际赢了/输了多少钱
- EV：基于当时的胜率，你应该赢/输多少钱

**例子：**
- 你All-in时胜率60%，底池10000
- 你的EV = +2000 (60% * 10000 - 40% * 10000)
- 实际结果：可能赢10000，也可能输0

如果长期来看，你的实际盈亏应该接近EV。差距大说明运气成分多。

## 安装使用

1. 用微信开发者工具导入项目
2. 选择测试号或填入自己的AppID
3. 编译运行

## 界面说明

- **记录**：牌桌上快速记录手牌
- **复盘**：查看历史手牌，填写详细信息和EV
- **统计**：数据分析，找出盈利/亏损模式
